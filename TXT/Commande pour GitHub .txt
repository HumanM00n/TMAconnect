Commande pour GitHub :

- Pour afficher tout les référentiels distants connectés :
	git remote -v

- Pour supprimer la connexion avec le référentiel distant :
	git remote remove origin


--------------------- PUSH LES INFOS --------------------- 

git add . --> Pour ajouter tout les fichiers 

git commit -m "" --> Permet d'enregistrer les modifications 

git psuh --> Pour envoyer les modifs vers Github
